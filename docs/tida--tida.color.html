tida.color

Color description module. Contains a structure for manipulating colors like RGB, there, and other ways to set colors. Mostly the RGB format is used, and the use of other formats will be converted to RGB.
Authors: TodNaz
License: MIT
enum PixelFormat: int;
Pixel format. Needed for conversion.
@safe Color!ubyte grayscale(ubyte gsc);
Converts grayscale to the library's standard color scheme.
Parameters:
ubyte gsc Grayscale value.
Example: 
auto color = grayscale(128);
Returns: RGBA
