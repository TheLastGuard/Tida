class Animation;
Object for describin animation.
bool isRepeat;
Whether the animation needs to be repeated.
@property @safe Image[] frames();
Animation frames.
@property @safe void frames(Image[] value);
Animation frames
@property @safe float speed();
Animation speed
@property @safe void speed(float value);
Animation speed
@safe void strip(Image image, int x, int y, int w, int h);
Creates frames from one sample picture.
Parameters:
Image image The picture from where the frames will be received.
int x The beginning of the cut by x-axis.
int y The beginning of the cut by y-axis.
int w Width frames.
int h Height frames.
Example: 
anim.strip(new Image("atlas.png"),0,0,32,32);
