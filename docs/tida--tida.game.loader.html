tida.game.loader

Resource loader module
Using the loader, you can load various resources that will be kept in memory. They can be accessed by their name or path, they can also be unloaded from memory.
Please note that it is unnecessary to reload download since it does not exist, it is implemented using the download-upload method. It will load the resource from the .temp folder.

License: MIT

A global resource loader instance. It is through it that you can download and receive resources.
class Loader;
Resource loader. Loads resources, fonts and more and keeps it in memory.
@safe T load(T)(immutable string path, string name = "null");
Will load the resource, having only its path as input. The entire loading implementation lies with the resource itself. The manager will simply keep this resource in memory.
Parameters:
T Data type.
string path Path to the file.
string name Name.
Retunrs: T
Throws: LoadException if the loader determines that the file does not exist. There may be other errors while loading, see their documentation, for example Image.
Example: 
Image img = loader.load!Image("a.png");
@safe T load(T)(immutable string[string] paths);
Loads multiple resources in one fell swoop using an associated array.
Parameters:
T Data type.
string[string] paths Paths and names for loading resources.
Throws: LoadException if the loader determines that the file does not exist. There may be other errors while loading, see their documentation, for example Image.
Example: 
loader.load!Image([
    "op1" : "image.png",
    "op2" : "image2.png"
]);
@trusted void free(T)(immutable string path);
Frees the resource from memory by calling the free construct on the resource if it has unreleased pointers and so on, and later removes the resource from the array, letting the garbage collector destroy this object.
Parameters:
T Resource class
string path Name or Path to file resource
Example: 
loader.free!Image("myImage");
@trusted void free(T)(T obj);
Frees the resource from memory by calling the free construct on the resource if it has unreleased pointers and so on, and later removes the resource from the array, letting the garbage collector destroy this object.
Parameters:
T Resource class
T obj Resource object
Example: 
auto myImage = loader.load!Image(...);
loader.free(myImage);
@trusted T download(T)(immutable string url, immutable string altname, string name = "null");
Load a resource from the internet using libcurl. Saves to the .temp folder. After, downloads it.
Parameters:
string url Url resource
string altname filename save
string name local name
Returns: T if all goes well and the resource is loaded. null if the resource is not found.
@safe T get(T)(immutable string name);
Returns a resource by name or path.
Parameters:
string name name resource(or path)
Returns: null if the resource is not found. If found, will return a T of the appropriate size.
@safe Font getFont(immutable string path, immutable int size);
Will return the font by path / name and size. The size is required.
Parameters:
string path Path to file or name.
int size Font size.
Returns: null if the resource is not found. If found, will return a Font of the appropriate size.
@safe Font loadFont(immutable string path, immutable int size, string name = "null");
Load a font of a specific size.
Parameters:
string path Path to file font
int size The size of the font by which you can then find the font. It will load a font of a certain size.
string name The short name by which the font can be found.
Throws: LoadException if the file is not found. Exception if something goes wrong while loading the font.
Returns: Font
@safe void add(Resource res);
Will add a resource that was not loaded through the manager. Please note that it must have a path and a name.
Parameters:
Resource res Resource.
